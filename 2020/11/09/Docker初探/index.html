  • Dockerfile命令总结

  1. run命令，用于在构建镜像时执行一些构建镜像需要执行的命令，比如复制一些文件到指定路径
  2. CMD ，定义了一些命令，用于在镜像启动后，要执行的一些命令
  3. expose，用于暴露端口
  4. COPY 从宿主机拷贝文件到容器中.拷贝文件的路径,是相对于build命令提供的context路径参数
