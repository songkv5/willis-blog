  • 变量与引号的作用

    • echo “$result”或”${result}”,变量按照原格式输出
    • echo $result 会单行输出
    • echo ‘$result’ 直接输出$result

  • 分号的作用

  • 算数运算

    • 使用expr。注意：expr只适用整数运算；expr后面的表达式，一定要用空格分开。使用带精度的计算，可以使用「bc」命令
    • 使用(()) 或 []计算
    • 算数比较运算
      运算符 说明 用法
      -eq 等于 [ 3 -eq $mynum ]
      -ne 不等于 [ 3 -ne $mynum ]
      -lt 小于 [ 3 -lt $mynum ]
      -le 小于或等于 [ 3 -le $mynum ]
      -gt 大于 [ 3 -gt $mynum ]
      -ge 大于或等于 [ 3 -ge $mynum ]

  • shell脚本的参数

    • 参数相关的元字符说明

      元字符 说明
      $# 传递到脚本的参数个数
      $* 以一个单字符串显示所有向脚本传递的参数
      $$ 脚本运行的当前进程ID号
      $! 后台运行的最后一个进程的ID号
      \$@ 与$*相同，但是使用时加引号，并在引号中返回每个参数。
      $- 显示Shell使用的当前选项，与set命令功能相同。
      $? 显示最后命令的退出状态。0表示没有错误，其他任何值表明有错误。

    • $n 代表传入的第n个参数

  • 条件判断：要用[]包围，且中括号前后要预留空格

  • 文件判断

    语法 说明 用法
    -e filename 如果 filename 存在，则为真 [ -e /var/log/file1 ]
    -d filename 如果 filename 为目录，则为真 [ -d /tmp/mydir ]
    -f filename 如果 filename 为常规文件，则为真 [ -f /usr/bin/grep ]
    -L filename 如果 filename 为符号链接，则为真 [ -L /usr/bin/grep ]
    -r filename 如果 filename 可读，则为真 [ -r /var/log/file1 ]
    -w filename 如果 filename 可写，则为真 [ -w /var/mytmp.txt ]
    -x filename 如果 filename 可执行，则为真 [ -L /usr/bin/grep ]
    filename1 -nt filename2 如果 filename1 比 filename2 新，则为真 [ /tmp/test-nt /etc/test ]
    filename1 -ot filename2 如果 filename1 比 filename2 旧，则为真 [ /etc/test -ot /tmp/fuck ]
